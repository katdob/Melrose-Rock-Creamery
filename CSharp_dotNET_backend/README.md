# C# .NET Backend

A C# .NET console application project.

## Getting Started

### Prerequisites
- .NET SDK (version 10.0 or later)

### Running the Application

```bash
# Restore packages
dotnet restore

# Build the project
dotnet build

# Run the application
dotnet run
```

## Project Structure

```
Csharp_dotNET_backend/
├── Csharp_dotNET_backend.csproj  # Project file
├── Program.cs                     # Main entry point
└── README.md                      # This file
```

## Building and Running

The project uses the modern .NET SDK with:
- Implicit usings enabled
- Nullable reference types enabled
- Target framework: .NET 10.0

## Development

Edit `Program.cs` to modify the application logic.
